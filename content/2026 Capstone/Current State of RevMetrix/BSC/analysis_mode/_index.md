### What's on this page

- **Motor graph** and **SmartDot graph** display loaded session signals (see [Motor and SmartDot Graph Widget](../motor_graph_widget/)).
- Analysis action buttons open detail dialogs:
  - Motor/SmartDot FFT
  - Motor/SmartDot 1st derivative
  - Motor/SmartDot 2nd derivative
  - Motor/SmartDot bandpass
  - Motor/SmartDot wavelet via [Wavelet Dialog](../wavelet_dialog/)
- **Save** submits current session data to cloud storage.

### How to use this page

1. Open from [Data View](../data_view/), [Diagnostic Mode](../diagnostic_mode/), or [Shot View](../shot_view/).
2. Choose the analysis operation matching your question.
3. In dialog windows, toggle relevant series and inspect highlighted values.
4. Repeat analyses as needed on the same loaded session.

### Warnings
> Do not close the app during save submission.
