### Welcome

Welcome to the **RevMetrix Ball Spinner Controller**.

This app is the primary interface for running shots, controlling motors for diagnostics, and reviewing saved session data.

### App Overview

- The top menu bar is used for **navigation** and **simulation management**.
  - The **Navigation** menu is used to return to Home or open [Cloud Test](../cloud_test/).
  - The **Simulation** menu allows switching between real and simulated motors.
    - On desktop builds, only simulated motors are available.
- **[Shot Mode](../shot_mode/)** is used to build and run shot profiles.
- **[Diagnostic Mode](../diagnostic_mode/)** is used for manual motor control and diagnostics.
- **[Data View](../data_view/)** is used to load and analyze previously saved sessions.
