### Before you start

Run this checklist before using Shot Mode, Diagnostic Mode, or Data View workflows.

### Pre-run checklist

1. Confirm **E-Stop** is reachable
2. Verify the arm is in the proper orientation.
3. Confirm the cover is in place.
4. Confirm both SmartDots are inserted into the arm.
