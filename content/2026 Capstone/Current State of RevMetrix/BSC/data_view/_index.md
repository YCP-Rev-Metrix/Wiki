### What's on this page

- **Filter text box** searches table content.
- **Session type combo** filters All / Diagnostic / Shot.
- **Date/time range controls** define query window.
- **Search** loads sessions in range; **View All** loads all available sessions.
- **Session table** lists ID, timestamp, name, and shot/diagnostic type.
- **Analyze** opens [Analysis Mode](../analysis_mode/) for selected session.
- **Replay** opens [Shot View](../shot_view/) for selected session.
- **Edit** opens [Shot Mode](../shot_mode/) populated from selected shot session.

### How to use this page

1. Set filters and date/time bounds.
2. Press **Search** or **View All**.
3. Select exactly one row in the table.
4. Choose **Analyze**, **Replay**, or **Edit** depending on the task.

### How do the filters work

- The only filters that query the database are the **start** and **end** date/time controls.
- The other filters operate on data that is already loaded in the table.
- If you change the date range, you must press **Search** again to fetch updated results.
- The **Filter** text box searches across all displayed table fields.
- Instruction points used for editing are loaded in the row data but are hidden from display.
