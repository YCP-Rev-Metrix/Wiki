## Registration Page
The registration page allows users to fill in a number of fields to create a new account. 
These fields are username, password, password confirmation, first name, last name, email, email confirmation, and phone number. 
Once all of these fields have been filled in, the user can click register to create the account and send them back to the home page to login.
